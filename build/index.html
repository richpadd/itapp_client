Project Background & Assets

Welcome! I have recently been curating an IT training course aimed at non-technical professionals. One planned feature was to provide additional online reference material to course participants, including a Glossary of IT terms that are covered during the course.

This project provided me with an excellent opportunity to try to build a dynamic IT Terms Glossary as opposed to just a static webpage. A database of IT Terms would be established and accessed via APIs to a read-only Glossary page, and a separate CRUD application built to administer the data. To maximise the benefits of this database & API approach, I also proposed a Quiz feature, where users can be asked to choose the correct IT term definition from a list.

From this concept the user requirements were created, and a project initiated to prototype, build and ultimately implement the solution on cloud, in line with the project rubric. Full details of which are covered in the submitted project document.

The result is this IT Terms website. The related webpages and artifacts can be accessed from the buttons below, and some via the navigation in the header.

> IT Terms Admin Console (CRUD app built with React.js - Password in Project document) > Glossary of IT Terms Page (built with Vue.js) > IT Terms Quiz (built with plain JavaScript) > GitHub (Client repo and API server node.js repo) > API Log Page (Access has been provided to the API logs) > Summary Video (In the - hopefully - unlikely event any functions are unavailable!)

Further information regarding the User Requirements capture (JIRA), UI Design, Database design, Infrastructure Setup, CI/CD implementation can all be found in the submitted project document.
